# Website-BubbleBear

Welcome to the BubbleBear website !
The goal of this website is for users to compete against each other by playing the game BubbleBear.

Features : 
Sign up and login :
- Users are able to create an account by providing an email adress, a username and a password.
- Once a account is created you can use the provided data to login whenever needed.

Playing the game : 
Users can play BubbleBear both online and offline.
- When playing online, your scores are automatically recorded on the leaderboard.
